# Mom Test Interview Template

Based on Rob Fitzpatrick's *The Mom Test*. The core idea: talk about the user's life, not your idea. Ask about specific past behavior, not hypothetical future intent. Avoid questions your mom would answer yes to just to be nice.

## How to use this template

Before any project, run 3-5 interviews with people who match your target persona. Use one file per interview (e.g. `docs/interviews/2026-01-15-jane-doe.md`).

## Good questions (use these)

Rank the ones that fit your product. Skip questions that don't.

- "Tell me about the last time you <did the task your product addresses>."
- "What did you do right before that? What did you do right after?"
- "What was the hardest part?"
- "Why was that hard?"
- "What have you tried to solve it? How did that go?"
- "How much time/money does this cost you per <week/month>?"
- "Who else does this? Can you introduce me?"
- "What are you using instead today?"
- "What would have to be true for you to <pay / switch / recommend>?"

## Bad questions (avoid)

- "Would you use a product that <description>?" (everyone says yes to be nice)
- "Do you think it's a good idea?" (not their job to judge)
- "How much would you pay for it?" (hypothetical, useless)
- "Would you buy it if it did X?" (same problem)
